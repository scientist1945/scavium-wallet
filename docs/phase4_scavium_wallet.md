# SCAVIUM Wallet — Phase 4 Documentation

## 🧭 Overview

Phase 4 expands the wallet with **asset management and ERC-20 support**.

---

## 🪙 Token System

### token_registry_controller.dart

Handles:

- token list
- persistence
- add/remove tokens

---

## 📦 Asset Model

Represents:

- native asset
- ERC-20 tokens

---

## 💰 Balances

### assets_controller.dart

Aggregates:

- native balance
- token balances

---

## 💸 ERC-20 Transactions

### send_token_controller.dart

Supports:

- contract calls
- decimals handling
- transaction execution

---

## ➕ Add Token

### add_token_screen.dart

Allows:

- input contract address
- fetch metadata
- register token

---

## 📊 Result

Phase 4 delivers:

- full token support
- asset aggregation
- ERC-20 transfers

---

## ⚠️ Limitations

- no history yet
- no auto-refresh
- manual updates only

---

## 🚀 Conclusion

Phase 4 turns the wallet into a **multi-asset platform**.
